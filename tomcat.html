O que o Apache?

O Apache: Um dos Servidores Web Mais Populares

O Apache é um software de código aberto amplamente utilizado como servidor web. Sua função
principal é servir conteúdo para a internet, ou seja, quando você digita um endereço
de site em seu navegador, é muito provável que ele esteja sendo servido por um servidor Apache.

O que é Tomcat?

O Tomcat: Um Container de Servlets Java

O Tomcat é um container de servlets de código aberto, baseado em Java. Isso
significa que ele é um software que permite executar aplicações web escritas
em Java, mais especificamente utilizando tecnologias como Servlets e JSPs.
